Global Integrity
================

## Requeriments

* Node 0.10+
* Ruby 1.9.3+

Install globally dependencies:

    npm install -g grunt-cli bower


## To install

    bundle install
    npm install
    bower install

## To run

On development
    
    grunt [watch]

On production
    
    grunt build    

# Deploy

    bundle exec cap [staging|production] deploy

# gh-pages

    grunt gh-pages
